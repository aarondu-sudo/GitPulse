# GitHubStar

发现 GitHub 上最热门的项目

## 功能

- 多维度热门：Trending 排行、Star 增长最快
- 语言筛选：支持 10+ 种编程语言
- 时间筛选：今日/本周/本月
- 关键词搜索：快速找到感兴趣的项目
- 响应式设计：支持桌面、平板、手机

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 数据来源

- [GitHub Trending API](https://github.com/hyr0n/py-Github-Trending-Api)
- [GitHub Search API](https://docs.github.com/en/rest/search)
